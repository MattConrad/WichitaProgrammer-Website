Cold Fusion   Javascript   Microsoft SQL Server

This project used Javascript to enhance a survey feature of a web application. The original version of the survey displayed different types of questions (radio button group, yes/no question, text entry) in a vertical form to be answered one at a time. After the enhancements, answers to one question could create ripple effects through the rest of the survey. An answer to a question could cause other questions to be answered automatically, hide questions from view, reveal questions that had previously been hidden, or combinations of these effects. This feature also included an admin UI so the dynamic effects could be defined and managed by end users.
